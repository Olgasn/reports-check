**Инструкция по установке и настройке**

В начале:

1. Установить NodeJS
2. После установки ввести в cmd: npm install --global yarn

Запуск сервера:

1. В cmd перейти в папку server
2. Ввести команду: yarn install
3. После завершения запустить: yarn start:dev

Запуск клиента:

1. В cmd перейти в папку client
2. Ввести команду: yarn install
3. После завершения запустить: yarn dev
